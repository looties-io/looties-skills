# google-ai-seo-optimization

Google-specific SEO guidance for AI Overviews and AI Mode readiness, based on
Google's AI optimization guide.

Use this skill for audits and implementation work around:

- Googlebot access, crawlability, indexability, and canonicals
- snippet eligibility and noindex boundaries
- JavaScript SEO and rendered content checks
- structured data accuracy
- Google AI SEO mythbusting

The skill treats AI Search readiness as strong SEO work, not as a separate bag of
AI-only tricks.

Source checked: 2026-05-17

Primary sources:
- https://developers.google.com/search/docs/fundamentals/ai-optimization-guide
- https://developers.google.com/search/docs/appearance/ai-features
