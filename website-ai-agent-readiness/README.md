# website-ai-agent-readiness

Website readiness guidance for AI agents, answer engines, browser agents, and
direct machine readers.

Use this skill for audits and implementation work around:

- `llms.txt` and `llms-full.txt`
- Markdown site mirrors such as `index.md`
- OKF / Open Knowledge Format bundles
- machine-readable pricing such as `pricing.md`
- `ai-index.json`, `entities.json`, and public entity datasets
- `robots.txt`, `sitemap.xml`, and static asset headers
- SEO smoke tests for agent-readable files

The skill is intentionally separate from the Google AI SEO skills. It can help
ship useful agent-readable assets, but it does not treat `llms.txt`, OKF,
Markdown mirrors, or machine-readable pricing as Google Search requirements.

Source checked: 2026-06-16

Primary context:
- Marketing Skills v2.4.2 OKF update
- Google's public guidance that AI Overviews and AI Mode use normal Google Search
  SEO foundations, not special AI-only files
